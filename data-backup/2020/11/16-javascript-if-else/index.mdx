decription: If-else is a common and very important programming concept that helps developers to take an action if a condition is met or another action if not met.
tags: ["javascript"]
---

`if...else` in JavaScript helps us to make decisions.

```javascript
if (age > 60) {
  console.log("Senior");
} else {
  console.log("Not Senior");
}
```
