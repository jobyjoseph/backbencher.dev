As a fullstack JavaScript developer, I come across different technologies and tools on a daily basis. This is a collection of such tools that might come handy for you as well.

<Olink link="https://expressjs.com/">Express</Olink> - A Nodejs framework to create
websites and APIs easily.
<br />
<Olink link="https://www.npmjs.com/package/express-async-errors">
  ExpressJS Async Errors
</Olink> - NPM package to enable Express framework to handle asychronous errors.
<br />
<Olink link="https://express-validator.github.io">Express Validator</Olink> - Validate
or sanitize API requests received by an Express application.
<br />
<Olink link="https://helm.sh/">Helm</Olink> - Package manager to find, share and
use software built for Kubernetes.
<br />
<Olink link="https://istio.io/">Istio</Olink> - Platform independent service mesh
that comes handy in microservice development.
<br />
<Olink link="https://jestjs.io/">Jest</Olink> - JavaScript testing framework.
<br />
<Olink link="https://mongoosejs.com/">Mongoose</Olink> - NPM package to communicate
with MongoDB.
<br />
<Olink link="https://nodejs.org">Node.js</Olink> - A software using which we can
run JavaScript in a server.
<br />
<Olink link="https://www.postman.com/">Postman</Olink> - Software to test APIs.
<br />
<Olink link="https://www.npmjs.com/package/supertest">Supertest</Olink> - NPM
package used for testing purpose to send fake requests to APIs.
<br />
<Olink link="https://www.typescriptlang.org/">TypeScript</Olink> - Helps to define
data type of a variable while writing JavaScript.
