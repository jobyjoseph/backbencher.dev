By default, when we use `AppBar` component from Material UI, there is a thick box-shadow.

<Image src={imageDefaultElevation} alt="Default AppBar elevation" />

We can control the `box-shadow` using `elevation` attribute. Setting the value of `elevation` to `0` removes the box shadow.

```javascript
<AppBar elevation={0}>//...</AppBar>
```

Here is how the AppBar looks from elevation 0 to 4. I have set the background to transparent to easily see the elevation difference.

<Image src={imageMultipleElevations} alt="Multiple Elevations" />
